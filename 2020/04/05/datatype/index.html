Data Types

Author: Yilin Date: April 5, 2020  18:28:19 Category: Data Indicators

1. Introduction

Data type is very important for you to understand your dataset. You need to deal with different types of data and transfer them to be applicable to you machine learning models. Basically, for the currenly machine learning models all need numeric data. To understand the concepts of the data type will give you a clear understanding of your dataset, just like descriptive statistics methods.

2. Data Types

Data Types - 1

There are four different data types: Nominal, Ordinal, Interval and Ratio.

  1. Nominal: For myself, I regard it as a categorical data type. You have the specific information for this data type without any numeric values to describe it. For example, gender (male/female) and color (red/yellow/blue/green/…). But we could transfer them to numeric value such as dummy value (0/1) or numeric values (1,2,3,4) without any meaning for size or order.

  2. Ordinal: Could be described by numeric and non-numeric data but can’t do the arithmetic (addition, subtraction, multiplication and division). But we could know its rank or order from the data. For example, the stars customers give to the products (from 1 to 5) or how you want your steak cooked (rare, medium rare, medium and well down).

  3. Interval: It only could be described by numeric data. You could know the order and exact the distance between the data. But could only do the addition and subtraction for this kind of data. The classic example of an interval scale is Celsius temperature because the difference between each value is the same. For example, the difference between 60 and 50 degrees is a measurable 10 degrees, as is the difference between 80 and 70 degrees.

  1. Ratio: It also could be only described by numeric data. You could do any of the meaningful arithmetic. For example, height, weight and duration.

Data Types – 2

I also see from other websites and blogs, some of them define the data types into two parts: qualitative and quantitative.

Qualitative: we could regard it as categorical data or nominal data that supposed not be described as numeric data.

Quantitative: it includes two different types of numeric data discrete and continuous.

Discrete data: it’s only could be integer instead of number with decimal. Usually, it is used to count something. For example, count how many apples.

Continuous data: like its name, it usually to describe the continuous number like height, weight and duration.

3. Summary

This post introduces what is the data type. Combines with the knowledge of descriptive statistics, you basically have a very clear understanding and what kind of problems for the dataset. You could start to pre-process the dataset and make this dataset applicable to your inferencial statistics methods and machine learning models.
